# Laborator Nr. 6: Introducere si initiere in Git si GitHub

## Scop
Initializarea unui repository Git, urmarirea modificarilor, lucrul cu branch-uri, merge si publicare pe GitHub.

## Ce am facut
- am creat repository local
- am adaugat un fisier readme.md
- am facut commit
- am conectat repository-ul la GitHub si am dat push
- am creat branch-ul dezvoltare, am modificat fisierul si am facut commit
- am facut merge in main si push final

## Comenzi folosite
git init, git status, git add, git commit, git log, git branch, git 
checkout, git merge, git push
